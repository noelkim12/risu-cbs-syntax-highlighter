Usage

이 라이브러리를 사용하려면 단 한 줄의 코드면 충분합니다:

<!-- 1. 스크립트 로드 -->
<script src="cbs-editor.js"></script>

<!-- 2. 초기화 -->
<script>
  const textarea = document.getElementById('my-textarea');
  const editor = new CBSTextarea(textarea, {
    highlightEnabled: true
  });
</script>
